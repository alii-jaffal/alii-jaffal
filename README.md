## Featured Projects

### AI News Aggregator

An end-to-end AI news pipeline that collects AI-related updates from multiple sources, processes content, stores structured data, generates ranked story digests, and prepares personalized daily email newsletters.

- Built a Python pipeline for ingestion, enrichment, clustering, ranking, and newsletter generation.
- Used PostgreSQL with SQLAlchemy and Alembic for structured storage and migrations.
- Integrated Google Gemini for AI-generated story digests.
- Developed a FastAPI backend and React/TypeScript dashboard for browsing history, monitoring failures, and triggering safe reruns.

**Tech Stack:** Python, FastAPI, React, TypeScript, PostgreSQL, SQLAlchemy, Alembic, Docker, Gemini

---

### Customer Support RAG Chatbot

An AI-powered customer support chatbot that answers user questions using retrieval-augmented generation and creates support tickets when human help is needed.

- Built a RAG workflow for document-based answering.
- Used vector search concepts to retrieve relevant information.
- Added fallback logic for low-confidence or unresolved questions.
- Designed the system to be extendable into a backend service.

**Tech Stack:** Python, FastAPI, PostgreSQL, pgvector, embeddings, RAG

---

### University Management System

A full-stack university platform with role-based workflows for admins, teachers, and students.

- Built role-based features for announcements, student data, and marks management.
- Connected a React frontend with a PHP backend.
- Used SQL Server and T-SQL for database storage and queries.
- Designed workflows based on real university operations.

**Tech Stack:** React, PHP, SQL Server, T-SQL, JavaScript, HTML, CSS

### Wabubu Agent

An AI assistant that retrieves and reasons over real-world data such as weather and sports information.

- Integrated external APIs for dynamic responses.
- Used Gemini function calling to connect the assistant with tools.
- Added FAISS-based memory for improved contextual behavior.
- Designed structured agent logic for tool-based responses.

**Tech Stack:** Python, Gemini, FAISS, APIs, tool calling
